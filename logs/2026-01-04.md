## put up（上に置く/掲げる）

### 物理的に立てる
| 例 | 意味 |
|----|------|
| put up a billboard | 看板を立てる |
| put up a tent | テントを張る |
| put up a sign | 標識を立てる |
| put up a poster | ポスターを貼る |

### put up + 抽象名詞（示す、発揮する）
| 表現 | 意味 |
|------|------|
| put up resistance | 抵抗を示す |
| put up a fight | 戦いを挑む、抵抗する |
| put up a struggle | 抵抗する |
| put up a front | 見せかける、虚勢を張る |
| put up a case | 主張を展開する |
| put up an argument | 議論を展開する |

### put up + お金（出す、提供する）
| 表現 | 意味 |
|------|------|
| put up cash | 現金を出す |
| put up money | 金を出す |
| put up the funds | 資金を提供する |
| put up collateral | 担保を出す |

### put up + 前置詞
| 表現 | 意味 |
|------|------|
| put up with | 我慢する |
| put up for | 候補に挙げる、売りに出す |
| put up at | 〜に泊まる |
| put up to | そそのかす、けしかける |
| put up against | 対抗させる |

### put up のコアイメージ
```
put up = 上に置く/掲げる
    │
    ├─ 物理的に立てる（billboard, tent, photo）
    ├─ 抵抗を示す（fight, resistance）
    ├─ 主張する（case, argument）
    └─ 金を出す（cash, money）
```
