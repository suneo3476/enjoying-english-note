## put down のまとめ

| 用法 | 例 |
|------|-----|
| 倒す、始末する | put down for good |
| 安楽死（楽にする） | put down like an animal |
| 書き留める | put down my race |
| 文章を書く | put down sentences |
| 記録する | put down drunk and disorderly |
| 録音する | put down tracks |
| 予定に入れる | put down for a trial |

### コアイメージ
```
put down = 下に置く
    │
    ├─ 下に倒す → 始末する、安楽死
    ├─ 紙の上に置く → 書く、記録する
    ├─ テープに置く → 録音する
    └─ リストに置く → 予定に入れる
```

## 質問と回答

**Q: 安楽死のニュアンスはどうやって生まれた？**
- put down 自体に「安楽」の意味はない
- 下に置く → 横たえる → 眠らせる → 命を終わらせる
- 「殺す」を直接言わない婉曲表現
- 日本語の「楽にしてやる」がちょうど良い意訳

**Q: 書く系のニュアンスはどう生まれた？**
- ペンを紙の上に「置く」動作から
- インク/文字を紙に「置く」感覚
- 紙は机の上 = 下方向なので down がつく
- 録音の put down tracks も同じ（音をテープに「置く」）
