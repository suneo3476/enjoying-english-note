## ⏰ 1日のスケジュール

### 🚌 行きバス（30分）— Epop

**なぜここでEpop？**
- 朝は頭がフレッシュ → 能動的学習に最適
- タップだけでできる
- ゲーム感覚で目が覚める

**やること：**
```
1. 単語クイズ（15分）
   - 穴埋め問題をサクサク解く
   - 間違えた単語は「解き直しノート」に自動保存される

2. 文法学習（10分）
   - 1日1トピックずつ進める

3. リーグ戦チェック（5分）
   - ランキング確認でモチベ維持
```

---

### 🏢 昼休み・個人ブース（30分）— 声出しメイン

**なぜここで声出し？**
- 1日で唯一「外で声を出せる」貴重な時間！
- 集中できる環境

**パターンA：Breaking Bad 音読（推奨）**
```
1. 【視聴】1シーン見る（字幕なし）（5分）
2. 【確認】Language Reactorで英語+日本語字幕確認（5分）
3. 【音読】セリフを声に出して3回読む（10分）
4. 【シャドーイング】音声に合わせて同時に読む（10分）
```

**パターンB：Claudeと英会話**
```
1. 英語でテキストチャット3往復（15分）
2. 添削フィードバックをもらう（10分）
3. 改善版フレーズを声に出して練習（5分）
```

**💡 週のバランス例：**
- 月・水・金 → パターンA（Breaking Bad）
- 火・木 → パターンB（英会話）

---

### 🚌 帰りバス（30分）— リラックス視聴

**なぜここで動画？**
- 仕事終わりで疲れてる → 受動的インプット
- 「見るだけ」で罪悪感なし

**オプション1：FuwaMoco切り抜き**
```
- YouTube字幕ON
- 5-10分の切り抜きを2-3本
- 気になるフレーズがあればスクショ
```

**オプション2：Breaking Bad 予習/復習**
```
- 明日音読するシーンを先に見ておく（予習）
- または昼に音読したシーンを再視聴（復習）
- Netflixアプリで事前ダウンロードしておくとオフラインOK
```

**オプション3：疲れすぎた日**
```
- FuwaMocoのフル配信をぼーっと見る
- 字幕ONにしておけば、それだけで学習になる
- 寝落ちしてもOK！
```

---

### 🏠 自宅（30分）— まとめ & じっくり

**なぜここでまとめ？**
- 落ち着いた環境で整理できる
- 1日の学習を定着させる

**オプションA：その日の振り返り（推奨）**
```
1. 今日覚えた表現を3つメモに書く（10分）
2. Breaking Badで出てきたフレーズ復習（10分）
3. Epopの「解き直しノート」確認（10分）
```

**オプションB：Language Reactor 精読**
```
1. Breaking Badの1シーンを徹底分析（20分）
   - 知らない単語を全部調べる
   - 文法構造を確認
2. 音読練習（10分）
```

**オプションC：別コンテンツで気分転換**
```
- 別のNetflixドラマ（Better Call Saulなど）
- Tech系YouTube（Fireship、ThePrimeagen）
- 技術ドキュメントを英語で読んでみる
```

**オプションD：疲れた日**
```
- FuwaMoco配信を見ながらゴロゴロ
- 字幕ONにしておけばOK
- 0分よりマシ！
```
